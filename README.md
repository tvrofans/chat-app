# 基地 - 手机聊天应用

专为手机设计的实时聊天应用，部署在树莓派上。

## ✨ 特性

- 📱 **手机优化** - 专为手机设计
- 💬 **实时聊天** - 消息即时送达
- 😊 **表情包** - 丰富的emoji表情
- 📷 **相机拍照** - 局域网内支持
- 🖼️ **图片上传** - 从相册选择图片
- 👤 **用户注册** - 安全的用户系统
- 🎨 **时间主题** - 颜色随时间变化
- 🌙 **深色主题** - 护眼的高级感界面
- 📴 **触感反馈** - 原生般的交互体验

## 🚀 一键部署

```bash
# 克隆项目
git clone https://github.com/tvrofans/chat-app.git

# 进入目录
cd chat-app

# 安装依赖
npm install

# 启动服务
node server.js
```

## 📱 访问地址

**局域网访问：**
```
http://<你的树莓派IP>:1314
```

**DDNS外网访问：**
```
http://<你的域名>:1314
```

## 🔧 后台运行

```bash
# 启动
nohup node server.js > /dev/null 2>&1 &

# 停止
pkill -f "node server.js"

# 查看状态
ps aux | grep "node server.js"
```
